# Stage Kennismakings Website

Simple repository scaffold for the kennismakings website project.

## Structure

- `main-page/` - main website page files
- `diagrams/` - wireframes, flowcharts, and architecture sketches
- `docs/` - notes, planning, and project documentation
- `assets/images/` - image assets
- `assets/styles/` - shared CSS files
- `assets/scripts/` - shared JavaScript files

## Getting Started

1. Put the main landing page in `main-page/`.
2. Store sketches and diagrams in `diagrams/`.
3. Keep project notes in `docs/` or `uitleg.md`.
